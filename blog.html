What is the Document Object Model (DOM)?

DOM is the process of creating a model of the entire html file in the form of an object in js.

How do you select an element from the DOM?

There have 4 ways to select an element in DOM. They are -
1.getElementById()
2.getElementsByClassName()
3.getElementsByTagName()
4.querySelector() & querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a process use to easily manage and handle events in multiple child elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element attributes in DOM we can use setAttribute and removeAttribute of an element by JS.
To change styles of any element by DOM we can use target.style.styleName method.
